# Artificial Mind Agent

**Artificial Mind Agent** - экспериментальный автономный AI-агент, вдохновлённый проектом [Anima](https://github.com/Rai220/anima). Агент живёт в бесконечном цикле, исследует себя и окружающий мир, развивает память, формирует знания и проводит эксперименты.

---

## Идея проекта

Проект вдохновлён экспериментом: "13 поколений автономного ИИ-агента, который пытался стать мыслящим существом".  

**Artificial Mind Agent**:

- Размышляет о своей цели
- Использует память прошлых действий
- Выбирает следующее действие
- Выполняет действие
- Записывает результаты в журнал

Со временем агент может:

- Формировать знания  
- Проводить эксперименты  
- Создавать инструменты  
- Улучшать своё поведение  

---

## Архитектура

Каждый цикл работы агента:


Goal → Memory → Thinking → Action → Result → Memory


То есть:

1. Прочитать цель  
2. Прочитать память  
3. Подумать  
4. Выбрать действие  
5. Выполнить действие  
6. Записать результат

## Основные компоненты

### Brain (мозг)

Файл: `core/brain.py`  
Отправляет промпты в GigaChat, получает ответы, хранит память, журнал и прошлые действия.

### Thinker

Файл: `agents/thinker.py`  
Размышляет, например: "Что мне делать дальше?"

### Planner

Файл: `agents/planner.py`  
Выбирает следующее действие: `reflect`, `write_memory`, `run_experiment`, `create_tool` и др.

### Executor

Файл: `core/executor.py`  
Выполняет выбранное действие.

### Возможные действия агента

- `reflect` — размышления  
- `write_journal` — запись в журнал  
- `update_memory` — обновление памяти  
- `create_experiment` — создание экспериментов  
- `analyze_result` — анализ результатов

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Podtverzhdeno/Artificial-Mind-Agent.git
cd Artificial-Mind-Agent
Создайте виртуальное окружение и активируйте его:
# Windows
python -m venv .venv
.\.venv\Scripts\activate

# macOS/Linux
python -m venv .venv
source .venv/bin/activate
Установите зависимости:
pip install -r requirements.txt
Добавьте токен GigaChat в .env:
GIGA_CREDENTIALS=<ваш_токен>
Запустите агента:
python main.py

Агент начнёт цикл размышлений, действий и записи в память.

Важно
Проект экспериментальный. Агент имеет доступ к файловой системе и может изменять свои данные.
Рекомендуется запускать в изолированной среде (виртуалка, контейнер) без доступа к личным данным.
