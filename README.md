**Проект: Artificial Mind Agent**

Идея проекта вдохновлена реализацией: "13 поколений автономного ИИ-агента, который пытался стать мыслящим существом.". 

Ссылка на автора **https://github.com/Rai220/anima**

Artificial Mind Agent — это экспериментальный автономный AI-агент, который живёт в бесконечном цикле и пытается исследовать себя и окружающий мир.

Агент:

размышляет о своей цели
использует память прошлых действий
выбирает следующее действие
выполняет его
записывает результаты в журнал

Со временем агент может:

формировать знания
ставить эксперименты
создавать инструменты
улучшать своё поведение

Это лаборатория для экспериментов с автономными AI-системами.

Основная идея архитектуры

Каждый цикл работы:

Goal → Memory → Thinking → Action → Result → Memory

или

1. Прочитать цель
2. Прочитать память
3. Подумать
4. Выбрать действие
5. Выполнить действие
6. Записать результат

**Основные компоненты**

Brain (мозг)

Связь с GigaChat.

Файл:

core/brain.py

Он будет:

отправлять промпты
получать ответы
Memory

Хранит:

память
журнал
прошлые действия
core/memory.py
Thinker

Агент размышляет:

agents/thinker.py

пример:

"Что мне делать дальше?"
Planner

Выбирает действие.

agents/planner.py

например:

reflect
write_memory
run_experiment
create_tool
Executor

Выполняет действие.

core/executor.py
Возможные действия агента

Например:

reflect
write_journal
update_memory
create_experiment
analyze_result
Пример цели агента

config/goal.md

Ты — автономный агент.

Твоя задача — исследовать себя и окружающий мир.

Ты можешь:
- размышлять
- записывать мысли
- анализировать прошлые действия
- создавать эксперименты
- улучшать свои инструменты

Используй файловую систему как память.
Пример цикла агента
Cycle 1

Thought:
Я должен начать с понимания своей цели.

Action:
reflect

Result:
написал размышление в journal
